Everyday Expenses Tracker

Overview
The Everyday Expenses Tracker is a GUI-based Python application for managing and analyzing personal expenses. It provides users with an intuitive interface to add expenses, track their budget, and visualize spending trends with graphs.

Features:
-Expense Logging: Add expenses with category, amount, and optional description.
-Budget Monitoring: Alerts users when expenses exceed a predefined budget limit.
-Spending Analytics: Visualize expense trends using bar graphs.
-Data Export: Save expense data to a CSV file for further analysis.
-Persistent Storage: Uses SQLite database to retain expense records.

Installation & Setup
Ensure you have the following installed:
-Python 3.x
-Required Python libraries: `tkinter`, `sqlite3`, `pandas`, `matplotlib`, `datetime`
